AI & Machine Learning Project

Project Machine Learning berbasis Google Colab menggunakan dataset distributor untuk analisis data, prediksi, dan evaluasi model AI.

Project ini mencakup Exploratory Data Analysis (EDA), preprocessing data, modeling machine learning, dan evaluasi performa model.

Tools yang digunakan: Python, Pandas, NumPy, Matplotlib, Scikit-Learn, dan Google Colab.
